<p>Render the input text</p>

<h2>Component usage</h2>

```js
const [inputValue, setInputValue] = useState('');

<InputText
  label="Nome"
  placeholder="placeholder"
  value={inputValue}
  name="inputName"
  inputId="inputId_1"
  onChange={(e) => setInputValue(e.target.value)}
/>;
```
