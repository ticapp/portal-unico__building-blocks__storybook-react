<h2>Description</h2>

<p>
  Renders the language selector component. When it change, it will trigger the{' '}
  <b>onLanguageChange</b> callback
</p>

<h2>Component usage</h2>

<p>Using default args</p>

```jsx
<LanguageSelector />
```

<p>Using specific language list</p>

```jsx
<LanguageSelector languages={{ pt: 'PT', en: 'EN', es: 'ES' }} active={'pt'} />
```
