<h2>Description</h2>

<p className="fs-16">
  Renders the application logo, that has two forms. If in a homepage, logo does not have link to redirect else redirects always to homepage.
</p>
<p className="fs-16">Check all props available on Docs tab.</p>

<h2>Component usage</h2>

<label>With redirect to homepage</label>

```tsx
import { Logo } from '@ama-design-system/logo';

<Logo />;
```

<label>Without redirect to homepage</label>

```tsx
import { Logo } from '@ama-design-system/logo';

<Logo isHomepage={true} />;
```
