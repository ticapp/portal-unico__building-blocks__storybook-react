<h2>Description</h2>

<p>An abstraction that deals with the rendering of React-Router and NextJS links so it can be used on both platforms</p>

<h2>Component usage</h2>

<p>React-Router</p>

```js
<Link link={'#'}>hello world!</Link>
```

<p>NextJS</p>

```js
<Link link={'#'}>hello world!</Link>
```
