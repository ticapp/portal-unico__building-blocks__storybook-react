  <u>Link</u>
</h1>

<h2>Description</h2>

<p className="fs-16">An abstraction that deals with the rendering of React-Router and NextJS links so it can be used on both platforms.</p>
<p className="fs-16">Check all props available on Docs tab.</p>

<h2>Component usage</h2>

<label>React-Router</label>

```tsx
import { Link } from '@ama-react-design-system/link';

<Link link={'#'}>hello world!</Link>;
```

<label>NextJS</label>

```tsx
import { Link } from '@ama-react-design-system/link';

<Link link={'#'}>hello world!</Link>;
```
